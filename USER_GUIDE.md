# AI视频总结功能 - 用户使用指南

## 快速开始

### 1. 启动服务

**启动后端**:
```bash
cd backend
python app.py
```

**启动前端**:
```bash
cd frontend
npm run dev
```

### 2. 访问应用

打开浏览器访问: `http://localhost:5173`

## 功能说明

### AI总结功能

#### 1. 输入视频链接

支持的视频平台：
- **B站 (bilibili.com)**: 优先推荐，国内访问稳定
- **YouTube (youtube.com)**: 需要科学上网环境

在输入框中粘贴视频链接，系统会自动识别平台。

#### 2. 点击「开始AI分析」

点击后会自动执行以下操作：
1. **并行请求**: 同时发起所有数据请求
2. **字幕提取**: 尝试获取视频字幕
3. **ASR回退**: 如果没有字幕，自动使用音频转写
4. **AI总结**: 生成结构化视频总结
5. **思维导图**: 自动生成可视化导图
6. **数据缓存**: 所有数据缓存，切换Tab无需等待

#### 3. 查看结果

**视频总结Tab**:
- 📝 视频概述: 整体内容概括
- 📚 内容大纲: 按时间线组织的章节
- 💡 核心要点: 重要知识点提取
- ✅ 总结结论: 主要收获和建议

**字幕文本Tab**:
- 显示完整字幕/转写文本
- 支持关键词搜索
- 点击时间戳跳转到视频对应位置
- 支持展开/收起长文本

**思维导图Tab**:
- 交互式可视化思维导图
- 支持缩放和拖拽
- 可切换Markdown文本视图
- 支持导出为Markdown文件

**AI问答Tab**:
- 基于视频内容的智能问答
- 流式输出，实时显示回答
- 支持多轮对话
- 自动引用时间戳

## 使用技巧

### 技巧1: 快速获取视频信息

对于B站视频，直接复制链接粘贴即可：
```
https://www.bilibili.com/video/BVxxxxxxx
```

系统会自动：
- 获取视频标题
- 提取视频封面
- 尝试获取字幕

### 技巧2: ASR转写的使用

当视频没有字幕时，系统会自动使用ASR（语音识别）功能：

**ASR处理时间**:
- 短视频（<5分钟）: 约30-60秒
- 中等视频（5-15分钟）: 约1-2分钟
- 长视频（>15分钟）: 约2-5分钟

**ASR准确性**:
- 中文语音识别准确率>90%
- 背景音乐会降低准确性
- 多人对话场景可能需要人工校对

### 技巧3: 切换Tab查看缓存

所有数据在第一次点击「开始AI分析」时加载完成：

```
点击"AI分析" → 并行加载所有数据 → 数据缓存
     ↓
切换Tab → 直接显示缓存 → 无需等待
```

### 技巧4: 导出功能

**导出AI总结**:
- 格式: Markdown (.md)
- 内容: 完整的结构化总结
- 用途: 笔记、分享、存档

**导出思维导图**:
- 格式: Markdown (.md)
- 内容: 思维导图Markdown源码
- 用途: 导入到其他思维导图工具

## 常见问题

### Q1: AI分析需要多长时间？

**A**:
- 有字幕的视频: 10-30秒
- 无字幕需ASR: 1-5分钟（取决于视频长度）

### Q2: 为什么有些视频无法获取字幕？

**A**: 可能原因：
1. 视频本身没有字幕
2. B站API限制（部分视频需要登录）
3. YouTube网络访问问题

**解决方案**: 系统会自动使用ASR转写音频生成文本

### Q3: ASR转写的文本准确吗？

**A**:
- faster-whisper模型的中文识别准确率>90%
- 对于清晰的人声对话效果最好
- 音乐、噪声可能影响准确性

### Q4: 可以处理多长视频？

**A**:
- 理论上无限制
- 建议视频长度<30分钟（处理时间更短）
- 超长视频可能需要更长等待时间

### Q5: AI问答功能如何使用？

**A**:
1. 等待AI分析完成
2. 切换到「AI问答」Tab
3. 输入问题，按Enter发送
4. AI会基于视频内容回答

**推荐问题**:
- "这个视频主要讲了什么？"
- "视频中有哪些核心要点？"
- "你能详细解释一下视频内容吗？"

### Q6: 打字机效果是什么？

**A**:
- 在AI总结生成过程中，内容会逐字显示
- 模拟AI实时思考的效果
- 提升用户体验，让等待更有趣

### Q7: 如何提高AI总结质量？

**A**:
1. 选择有清晰语音的视频
2. 确保视频有字幕或语音清晰
3. 提供准确的视频标题和描述
4. 避免背景音乐过大的视频

- **存储**: 2GB可用空间
